#### Auto indexing
When you delete the value "nice" in a table like 

```
[
  0 => "lol"
  1 => "data"
  2 =>  "nice"
  3 =>  "seal"

]
```

To get :


```
[
  0 => "lol"
  1 => "data"
  2 =>  "seal"

]
```
